# Weixin Mini Program Server

## 必要信息
本地开发时
1. 系统启动会自动创建相关表，默认库名为：`weixin`
2. 项目所有表名均有`weixin_`前缀
3. 如不访问微信服务器，可使用`mocklogin`接口
4. 如需访问微信服务器，私信我要appId与appSecret
5. Just have fun
